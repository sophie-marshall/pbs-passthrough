a new way to watch.

powered by facial recognition

computer vision vector databases

powered by facial recognition, passthrough facilitates the retrieval and service or person-specific metadata within your video content. passthrough can help viewers enjoy an enhanced viewing experience by providing them with actor/actress names, bios, trivia, and more! inspired by Prime Video's X-Ray service, passthrough aims to provide comparable data using open source technologies.

at a specified frame rate, passthrough extracts frames and uses a multi-task cascaded convolutional neural network (MTCCN) to search for faces. in the provided image, two faces were detected by the MTCNN model, each accompanied by bounding box coordinates and a confidence score. identified faces are embedded using FaceNet and will eventually be used as query parameters.

for each embedded face, a query is sent to a database of known faces. the query compares the provided embeddings with its stored facial embeddings using the cosine similarity metric and returns the top 3 results along with any associated metadata. passthrough's vector search is suppored by PostgreSQL's pgvector extension.
